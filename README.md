# books-directory
NodeJS Rest Api for Books Directory

*Supported API:*
1. GET - Get all books from the directory
2. POST - Add a book to the directory
3. PATCH - Update a book by Id
4. GET - Get a book by id
5. DELETE - Delete a book by id

## Run Application:
`npm install`
`npm run build`
`npm run test`
`npm run start`
